Review of probabilities

Kolmogorov Axioms

  • \(0\leq P(E)\leq 1\)
  • \(P(S)=1\)
  • For any sequence of events \(E_1,E_2,\cdots\) that are mutually exclusive, then\[P(\cup_{i=1}^\infty E_i)=\sum_{i=1}^\infty P(E_i)\]

Inequalities

Bonferroni's Inequality

\[P(E\cap F)\geq P(E)+P(F)-1\]

Boole's Inequality

\[P(\cup_{i=1}^{\infty}E_i)\leq\sum_{i=1}^\infty P(E_i)\]

Distributions

Discrete Random Variables

Assumptions: 1. Independent trials: a specific event \(E\) in an experiment with probability \(p\) in a given trial 2. Repeating many times (\(n\) trials): success with probability \(p\), failure with probability \(1-p\)

Bernoulli distribution \(B(p)\)
  • Success: \(X=1\) or failure \(X=0\)
  • Probability mass function \[ p(0)=P\{X=0\}=1-p;\ p(1)=P\{X=1\}=p\ (0<p<1)\]
Binomial distribution \(Bin(n,p)\)
  • \(X\): number of success in \(n\) trials \[p(k)=P\{X=k\}=C_n^kp^k(1-p)^{n-k},\ k=0,1,2,\cdots,n\]
Geometric distribution \(G(p)\)
  • \(X\): number of trials until first success \[p(k)=P\{X=k\}=p(1-p)^{k-1},\ k=1,2,\cdots\]
Poisson distribution \(P(\lambda)\)
  • \(X:\{0,1,2,\cdots\}\) \[p(k)=P\{X=k\}=e^{-\lambda}(\lambda^k/k!),\ k=0,1,2,\cdots\]
Finite discrete distribution
  • \(X:\{1,2,\cdots,K\}\) \[p(k)=P\{X=k\}=N_k/N,\ k=1,2,\cdots,K,\ (N=N_1+\cdots+N_K)\]
  • Discrete uniform distribution: \[N_k=1,\ k=1,\cdots,K\]

Continuous Random Variables

There is a function \(f(\cdot)\) such that the cdf: \[F(x)=\int_{-\infty}^xf(u)\text du,\ \text{for all}\ x\in(-\infty,+\infty)\]

If \(F(\cdot)\) is continuous and piecewise differentiable, then - \(f(x)=\text d(F(x))/\text d x=F^\prime(x)\) - Probability density function: \(f(x)\)

Theorem A function \(f(\cdot)\) is a probability density function (pdf) of a continuous random variable if and only if 1. \(f(x)\geq 0\ \text{for all}\ x\in(-\infty, +\infty)\) 2. \(\int_{-\infty}^{+\infty}f(u)\text du=1\)
