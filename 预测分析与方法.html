时间序列基础

A sequence of random variables \(\{Y_t:t=1,2,3\cdots\}\) is called a stochastic process - It can serve as a model for an observed time series

特征

完整描述要用联合分布函数 \[F_{t_1,t_2,\cdots, t_n}(y_1,y_2,\cdots,y_n)=P(Y_{t_1}\leq y_1,Y_{t_2}\leq y_2,\cdots, Y_{t_n}\leq y_n)\]

随机过程的均值函数 \[\mu_t=\text E[Y(t)],\text{ for }t= 1,2,3,\cdots\] 自协方差函数 \[\gamma(t,s)=\text{Cov}[Y_t,Y_s]=\text E[(Y_t-\mu_t)(Y_s-\mu_s)]=E[Y_tY_s]-\mu_t\mu_s,\text{ for }t,s=1,2,3,\cdots\]

自相关函数 \[\rho(t,s)=\text{Corr}[Y_t,Y_s]=\cfrac{\text{Cov}[Y_t,Y_s]}{\sqrt{\text{Var}[Y_t]\text{Var}[Y_s]}}=\cfrac{\gamma(t,s)}{\sqrt{\gamma(t,t)\gamma(s,s)}}\ \text{ for }t,s=1,2,3,\cdots\]

常见模型

白噪音

A sequence of uncorrelated random variables \(\{w_t,t=1,2,3\cdots\}\) with mean 0 and finite variance \(\sigma_w^2\) - i.i.d. white noise: \(w_t\sim i.i.d (0,\sigma_w^2)\) - Gaussian white noise: \(w_t\sim N(0,\sigma_w^2)\)

\[\mu_t=\text E[w_t]=0\] \[\gamma(t+h,t)=\text{Cov}[w_{t+h},w_t]=\begin{cases} \sigma_w^2&h=0\\ 0&h\neq 0 \end{cases}\] \[\rho(t+h,t)=\begin{cases} 1 &\text{ if }h=0\\ 0&\text{ if }h\neq 0 \end{cases}\]

随机游走

Let \(e_1,e_2,\cdots\) be a sequence of independent, identically distributed random variables each with zero mean and variance \(\sigma_e^2\) The observed time series, \(\{Y_t,t=1,2,\cdots\}\) is constructed as follows: \[\begin{align} Y_1&=e_1\\ Y_2&=e_1+e_2\\ \vdots\\ Y_t&=e_1+e_2+\cdots+e_t\\ Y_t&=Y_{t-1}+e_t \end{align}\]

\[\mu_t=\text E[Y_t]=0\] \[\gamma(t,s)=\text{Cov}(Y_t,Y_s)=t\sigma_e^2\] \[\rho(t,s)=\cfrac{\gamma(t,s)}{\sqrt{\gamma(t,t)\gamma(s,s)}}=\sqrt{\cfrac{t}{s}}\]

移动平均

两点移动平均

\[Y_t=\cfrac{e_t+e_{t-1}}{2}\] \[\mu_t=\text E[Y_t]=0\] \[\gamma(h)=\begin{cases} 0.5\sigma_e^2&|h|=0\\ 0.25\sigma_e^2&|h|=1\\ 0&|h|>1 \end{cases}\] \[\rho(h)=\begin{cases} 1&|h|=0\\ 0.5&|h|=1\\ 0&|h|>1 \end{cases}\]

三点移动平均

\[Y_t=\cfrac{e_{t+1}+e_t+e_{t-1}}{3}\] \[\mu_t=0\] \[\gamma(h)=\begin{cases} \cfrac{3}{9}\sigma_e^2&h=0\\ \cfrac{2}{9}\sigma_e^2&|h|=1\\ \cfrac{1}{9}\sigma_e^2&|h|=2\\ 0&|h|>2 \end{cases}\] \[\rho(h)=\begin{cases} 1&h=0\\ \cfrac{2}{3}&|h|=1\\ \cfrac{1}{3}&|h|=2\\ 0&|h|>2 \end{cases}\]

平稳性

严平稳时间序列

The probabilistic behaviors of every collection of values \(\{Y_{t_1},Y_{t_2},\cdots, Y_{t_k}\}\) is identical to that of the time shifted set \(\{Y_{t_1+h},Y_{t_2+h},\cdots,Y_{t_k+h}\}\). That is\[P\{Y_{t_1}\leq y_1,Y_{t_2}\leq y_2,\cdots,Y_{t_k}\leq y_k\}=P\{Y_{t_1+h}\leq y_1,Y_{t_2+h}\leq y_2,\cdots,Y_{t_k+h}\leq y_k\}\]for all \(k=1,2,\cdots\), all time points \(t_1,t_2,\cdots,t_k\) all numbers \(y_1,y_2,\cdots,y_k\), and all time shifts \(h=0,\pm1,\pm2,\cdots\)

弱平稳时间序列

  • The mean function \(\mu_t\) is constant and does not depend on time \(t\)
  • The autocovariance function \(\gamma(s,t)\) depends on \(s\) and \(t\) only through their difference \(|s-t|\)

\[\mu_t=\mu\] \[\gamma(h)=\gamma(t+h,t)=\text{Cov}(x_{t+h},x_t)=\text E[(x_{t+h}-\mu)(x_t-\mu)]\] \[\rho(h)=\cfrac{\gamma(t+h,t)}{\sqrt{\gamma(t+h,t+h)\gamma(t,t)}}=\cfrac{\gamma(h)}{\gamma(0)}\]

参数估计

\[\bar y =\cfrac{1}{n}\sum_{t=1}^ny_t\] \[\hat\gamma(h)=\cfrac{1}{n}\sum_{t=1}^{n-h}(y_{t+h}-\bar y)(y_t-\bar y)\] \[\hat\rho(h)=\cfrac{\hat\gamma(h)}{\hat\gamma(0)}=\cfrac{\sum_{t=1}^{n-h}(y_{t+h}-\bar y)(y_t-\bar y)}{\sum_{t=1}^n(y_t-\bar y)(y_t-\bar y)}\] #### 证明

时间序列调整

类型 - 日期调整 - 总数调整 - 通货膨胀调整 - 数学变换调整

Calendar adjustments

调整统计时间长短不一致导致的偏误

方法：平滑一下

例如： \[W_t=Y_t\times\cfrac{所有月份天数的平均数}{当月天数}\] \[W_t=Y_t\times\cfrac{所有月份交易日的平均数}{当月交易日数目}\]

Population adjustments

调整总数不一导致的偏误

方法：除以总量，算单位量

例如：人均GDP

Inflation adjustments

调整通货膨胀带来的偏误

方法：以某年的CPI为参考

例如： \[x_t=y_t\times\cfrac{z_{2000}}{z_t}\]

Mathematical transformation

调整因规模变化带来的偏误，协整波动性

方法：取对数、取指数、Box-Cox

例如： \[w_t=\begin{cases} \log(y_t)&\text{if }\lambda = 0\\ (\text{sign}(y_t)|y_t|^\lambda-1)/\lambda &\text{otherwise} \end{cases}\]

时间序列分解

  • A trend pattern exists when there is a long-term increase or trend decrease in the data.
  • A seasonal pattern exists when a series is influenced by seasonal factors.
  • A cyclical pattern exists when the data exhibit rises and cyclical falls that are not of a fixed period.

\[Y_t=f(S_t,T_t,R_t)\] - \(Y_t\): Time series data at time \(t\) - \(T_t\): Trend-cycle component（趋势周期项）at period \(t\) - \(S_t\): Seasonal component（季节项）at period \(t\) - \(R_t\): Remainder component（残差项）at period \(t\)

加法分解：\(Y_t=S_t+T_t+R_t\) 乘法分解：\(Y_t=S_t\times T_t\times R_t\) （可以通过取对数转化为加法分解）

移动平均和Loess回归

用来平滑趋势周期项，减小随机方差

普通移动平均

\[\hat T_t=\cfrac{1}{m}\sum_{j=-k}^ky_{t+j}\] 其中，\(m=2k+1\)，叫做m-MA smoother

  • 要求m为奇数
  • 会出现首尾缺少数据的情况

例子： ![[预测分析与方法/263b98ce10159ba2bc8b09b61cd5c37.png]]

双重移动平均

做两次移动平均

2-point moving average of 4-point moving average (2 \(\times\) 4 MA) \[\begin{align} \hat T_{2.5}=\cfrac{y_1+y_2+y_3+y_4}{4}\\ \hat T_{3.5}=\cfrac{y_2+y_3+y_4+y_5}{4} \end{align}\]

  • Averaging these two 4 MA smoothers gives \[\hat T_3=\cfrac{1}{8}(y_1+2y_2+2y_3+2y_4+y_5)\]
  • 2\(\times\) 4 MA smoother is equivalent to a weighted moving average of order 5

也可以有其它阶数，反正就是两次移动平均

加权移动平均

\[\hat T_t=\sum_{j=-k}^ka_jY_{t+j}\] - \(m=2k+1,\sum_{j=-k}^ka_j=1,a_j=a_{-j}\)

Henderson's 5-, 7-, 9-, 13-, and 23-point weighted moving averages. - The greater the randomness, the larger the number of terms needed in the average. - The weights were calculated using the quadratic function

\[Q(j,m)=\begin{cases} (1-(\cfrac{j}{m})^2)^2&\text{for }-m<j<m\\ 0&\text{otherwise} \end{cases}\] \[a_j=\cfrac{Q(j,m)}{\sum_{i=-m}^mQ(i,m)}\]

Local regression smoothing（局部回归平滑）

回归局部趋势周期项

  • 假设这个观测区间内，趋势周期项为\(\hat T_t=a+b\cdot t\)
  • \(a\)和\(b\)是为了最小化加权平方和\[\sum_{j=-m}^ma_j(Y_{t+j}-a-b(t+j))^2\]
  1. Set the number of points to be used in the weighted regression.
  2. Weights are assigned to those data points.
  3. A line is fitted to the data with the values of \(a\) and \(b\) chosen to minimize the weighted sum of square.
  4. Repeat steps 1 to 3 for each observation.

Loess 局部加权回归平滑

上面的局部回归平滑的改良版，相当于根据残差调整权重，进行优化

Locally Weighted Scatterplot Smoothing (Loess) 1. An initial local regression is calculated, and the remainder component is calculated using \[\hat E_t=y_t-\hat T_t\] 2. The weights are adjusted so that observations with large errors receive smaller weights 3. Repeat steps 1 and 2 for several iterations to obtain the final estimate \(\hat T_t\)

分解方法

加法分解

For a time series with seasonal period \(m\) - 4 for quarterly data, 12 for monthly data, 7 for weekly data

    • If \(m\) is an even number, compute the trend-cycle component \(T_t\) using a \(2\times m\)–MA
    • If \(m\) is an odd number, compute the trend-cycle component \(T_t\) using a \(m\)–MA
    • Calculate the de-trended series (去趋势序列): \[y_t=\hat T_t\]
    • Average the de-trended values for that season to estimate the seasonal component for each season \(\hat S_t\)
    • The remainder component is calculated by subtracting the estimated seasonal and trend-cycle components: \[\hat R_t=y_t-\hat T_t-\hat S_t\]

乘法分解

For a time series with seasonal period \(m\) - 4 for quarterly data, 12 for monthly data, 7 for weekly data

    • If \(m\) is an even number, compute the trend-cycle component \(T_t\) using a \(2\times m\)–MA
    • If \(m\) is an odd number, compute the trend-cycle component \(T_t\) using a \(m\)–MA
    • Calculate the de-trended series (去趋势序列): \[y_t/\hat T_t\]
    • Average the de-trended values for that season to estimate the seasonal component for each season \(\hat S_t\)
    • The remainder component is calculated by subtracting the estimated seasonal and trend-cycle components: \[\hat R_t=y_t/(\hat T_t\hat S_t)\]
