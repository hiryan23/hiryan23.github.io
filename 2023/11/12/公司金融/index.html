公司金融好好学

本文最后更新于：17 天前

主要根据唐宗明老师的《公司金融》课程课件和上课内容整理。上《公司金融》课，上得非常爽。

总纲领

公司金融很重要，是考研科目（

公司金融定义和概况。1.1 什么是公司金融

公司金融核心就是估值。衡量公司价值是主要问题，伴随着估计项目、股票等（这个分类好随意）

给公司估值，可以用财务报表。预测未来的财务报表，然后计算无杠杆自由现金流，再估算WACC，折现算出公司价值。

公司如果关注现金流（只看价值部分），就可以看作多个项目的集合，因此可以用对项目的估值来给公司估值。 给项目估值，用现金流折现。 除此之外，给项目估值可以辅助公司决策，看要不要投资。用NPV和IRR方法

股权投资者关注公司的股票价值。因此公司股票也需要估值。有多种方法。

Lecture 1 公司价值原理

1.1 什么是公司金融

  • 公司金融与财务会计的区别
公司金融 财务会计
时间（轴） 未来 过去
目标 公司价值（未来现金流折现值） 公司盈利（过去一段时间的经营成果）
成本 机会成本 会计成本

  • 联系：未来是在以过去为基础上预计的

  • 资产负债表

    • 资本预算决策：应选择什么样的长期投资项目？（固定资产）
    • 资本结构决策：应该怎样为选定的项目筹措资金？（短期负债+长期负债+所有者权益）
    • 短期经营资产管理：怎样管理和筹措短期经营活动需要的资产？（流动资产-短期负债）

  • 现金流非常重要

  • 好的公司现金周期更短

    • 经营周期=存货周期+应收账款周期，显示企业完成一次经营活动的时间长度
    • 现金周期=经营周期-应付账款周期，显示企业在经营周期中占用自己现金的时间长度
    • 现金周期显示公司占用自身现金的时长，折射公司在产业链和行业竞争中的地位。

总结

  • 公司金融以财务会计为基础
  • 基于资产负债表模型，公司金融决策包括三大方面：投资、融资和营运资金管理
  • 公司金融决策中现金流很重要。但盈利是现金流的基础。

1.2 什么是公司

  • 企业的组织形式
    • 股份有限公司
      • 指全部资本分为等额股份，股东以其所持股份为限对公司承担责任，公司以其全部资产对公司债务承担责任的公司
    • 有限责任公司
      • 是指由一定人数股东组成的，股东以其出资额为限对公司承担有限责任，公司以其全部资产对公司债务承担无限责任的公司。
      • 非公开发行，因此不用公开财务会计信息等
    • 一人有限责任公司
      • 指只有一个自然人股东或一个法人股东的有限责任公司，公司的全部股份或出资全部归属于一个股东。
      • 每年要进行一次审计，证明公司财产独立于股东自己财产，否则要对债务承担连带责任。
  • 委托代理问题（两个）
    • 产生原因：信息不对称、目标不同、监督成本高

1.3 公司的目标

  • 股东财富最大化

  • 公司金融始于公司战略

    • 公司战略：
      • 产品市场
      • 资本市场
    • 对应公司金融决策：
      • 投资：新建、并购、分拆
      • 融资：债务权益融资、股利分配、控制权转让

  • 股东财富（股权的市场价值）等于股票数量乘以每股价格

  • 股权的内涵价值为归属股东未来现金流的折现值

  • 公司的内涵价值为公司未来自由现金流的折现值

  • 公司金融就是以股东财富最大化为目标，进行投资、融资、营运资金管理决策。股东财富表现为 股东权益的市场价值。所以，不论公司的投资决策，还是融资决策或股利、营运资金管理决策， 都要以增进股东财富为标准。

  • ESG

1.4 公司价值分析框架

  1. 公司的主营业务
    1. 商业模式与盈利模式：公司内涵价值的分析起点
    2. 公司治理（ESG）
  2. 行业供给与需求及未来趋势
    1. 行业供给及演变
    2. 行业需求及演变
    3. 行业的竞争格局
    4. 公司在行业中的竞争地位
  3. 公司的核心竞争力分析
    1. 公司所在的产业链分析
    2. 公司与竞争对手的差异
    3. 公司的核心竞争力
  4. 公司的财务分析
    1. 历史财务数据及演变趋势：盈利能力、偿债能力、资产周转率、流动性、成长性、现金流
    2. 从数据上验证前面的定性分析
    3. 为估值部分的收入拆分和预计提出假设提供依据
  5. 估值分析（DCF、比率方法）
    1. 假设与收入拆分
    2. 备考财务报表编制：利润表、资产负债表、现金流量表
    3. 终端价值估计
    4. WACC的估算：权益成本、债务成本、最优资本结构、无风险收益率、权益贝塔
    5. 比率方法：市盈率、或市销率、或EV/EBITDA等估值方法
    6. 敏感性分析
    7. 给出一个估值区间
  6. 投资风险分析
    1. 给不同的风险排序
    2. 风险化解的可能性为公司的核心竞争力
  • 价值分析的核心是找到价值驱动因素。通过不同因素之间的关系，最后计算得到公司的内涵价值。

总结

  • 公司价值决定于公司未来的现金流的折现值。拆分收入，编制备考财务报表、计算折现率、终端增长率假设等等都是估值关键环节
  • 公司估值是一个艺术。不同的人有不同的认知（定性分析），定性分析决定定量分析。所以，虽然方法一样，但认知不同决定了不同指标的预测值可能有偏差，估值结果有差异。
  • 公司与资本市场密切相关。尽管可以估计公司的价值（内涵价值），公司的股价或市值最终在资本市场被决定，从某个角度上说，资本市场为公司提供了“称重器”
  • 作为经济、金融活动主体必须遵循相关的法律法规约束，不合规就会带来风险和成本。一个具有投资价值的公司是考虑长远生存的，因此，ESG也是公司价值评估时考虑的重要方面。

Lecture 2 财务报表与自由现金流

2.1 财务报表阅读

  • 关注
    • 报表的结构等式
    • 每张报表的要点
    • 每张报表的重要（关键）科目
  • 为了
    • 验证公司价值定性分析中的：商业模式、行业竞争格局、公司在产业链中的地位、公司核心竞争力（波特五力模型）
    • 明确报表的关键指标，计算未来的自由现金流与哪些指标有关，寻找后续预测假设的依据

2.1.1 资产负债表

  • 营运资本
    • 净营运资本（NWC）=流动资产-流动负债
      • 通常随公司增长而增长
      • 反映企业用多少长期资本解决流动负债
    • 经营性营运资本（OWC）=经营性流动资产-经营性流动负债
      • 经营性流动资产：不包括货币资金，包括应收账款、应收票据、预付账款，存货等（即非现金流动资产）
      • 经营性负债不包括融资性负债（短期借款、长期借款、应付债券，这些负债都支付利息），包括应付票据、应付账款、预收账款，应付职工工资，应交税金和其它应付款（即不付息负债）
      • 反映企业在上下游业务中的谈判能力和议价水平
  • 应当注意：
    • 资产流动性（资产结构）
    • 负债水平与偿债能力、所有者权益的构成
    • 市价与账面价值（历史成本）
    • 核心资产与非核心资产
  • 关键科目举例
    • 资产方面
      • 货币资金、应收账款、预付账款、存货、固定资产、在建工程、无形资产、商誉
    • 负债和权益方面
      • 经营性负债：应付账款、预收账款（一般无利息）
      • 融资性负债（一般有利息）
      • 所有者权益

2.1.2 利润表

  • 注意
    • 公认会计准则（GAAP）影响利润的现金含量
    • 非现金项目（折旧、递延所得税）
    • 时间与成本（固定成本与变动成本）
    • 公司所得税（利息免税）
  • 利润指标
    • 毛利
      • 销售收入-营业成本（营业成本=产品销售成本+折旧）
    • 营业利润
      • 指提供产品和劳务产生的利润
      • 营业收入-营业成本-销售费用-管理费用
    • 息税前利润EBIT
      • 反映企业的核心业务盈利能力
      • 营业利润+其它利润
      • 净利润+所得税+利息（财务费用）
    • 利润总额（税前利润）
      • EBIT-利息费用+营业外收入-营业外支出
    • 净利润
      • 利润总额-所得税费用
    • 归属于母公司股东的净利润
      • 是在净利润基础上剔除少数股东损益。该指标是分析母公司股东权益价值的核心指标。比如EPS计算公式中的分子即为该指标
      • 归母净利润=净利润-少数股东权益
      • EPS=归母净利润/公司股份数
    • 综合收益总额
      • 净利润+其它综合收益（如可供出售金融资产中公允价值变动的部分，不计入当期损益，直接调整资本公积，但是这部分收益可以在综合收益总额中体现）

2.1.3 现金流量表

  • 现金流量表从三个方面解释现金变动原因

    • 经营活动
    • 投资活动
    • 筹资活动

  • 经营活动

    • 以净利润为起点，再调整非现金项目的影响和流动资产与流动负债（但不包括现金）的影响

  • 投资活动

    • 资本性资产的变化（固定资产的取得与出售）

  • 筹资活动

    • 权益和负债的变动

  • 经营活动现金流如果加利息，筹资活动现金流就要减利息（有点像费用资本化）

  • 现金流可以反映企业盈利质量的高低，比如经营活动现金流量指标除以净利润，可以看出净利润的经营性现金含量

2.2 自由现金流

2.2.1 自由现金流（FCF）的定义

  • 自由现金流是公司经营产生的现金流入扣除了保证公司持续经营所需要的现金流出以后，剩下的可以分配给投资人（股东和债券人）的现金流。
    • 经营性现金流（OCF）是公司持续经营下的现金流入，依赖自身的资产的生产和产品销售等来获得。
    • 持续经营需要在流动资产和固定资产上持续投入，具体包括：资本支出净额是公司保证正常生产经营在固定资产方面的投资净额；净营运资本变动是公司保证正常生产经营在流动资产方面的净投资。

2.2.2 自由现金流恒等式

\[CF(A)=OCF-CAPEX-\Delta NWC\]

  • 经营性现金流（OCF）（与现金流量表经营活动有区别，没考虑非现金流动资产与不付息负债）
    • =息税前利润（EBIT）+折旧（Dep.）-所得税
    • =息税折旧前利润（EBITDA）-所得税
    • =净利润+利息+折旧（假设所得税为0）
  • 资本支出净额（CAPEX）（与现金流量表投资活动一致）
    • =固定资产购买-固定资产出售
    • =期末固定资产净额-期初固定资产净额+折旧
    • =期末固定资产总值-期初固定资产总值
  • 净营运资本变动（ΔNWC）
    • =年末的净营运资本-年初的净营运资本
    • =（年末流动资产-年末流动负债）-（年初流动资产-年初流动负债）
    • =（年末流动资产-年初流动资产）-（年末流动负债-年初流动负债）
    • =流动资产变动额-流动负债变动额

2.2.3 股权自由现金流FCFE/CF（S）

\[CF(A)=CF(B)+CF(S)\]

  • CF（B）=利息（I）-净债务融资金额（DebtIR）

    • DebtIR=债务融资-债务归还

  • CF（S）=OCF-CAPEX-ΔNWC-I+DebtIR

  • 实际中，因为现金和有价证券是非经营性资产，预测很困难，所以用经营性资产产生的现金流（FCF）估值，具体为FCF=OCF-CAPEX-ΔOWC

  • 这个东西的折现值叫做公司的企业价值（EV）

    • 公司价值=负债+股东权益
      • =初始现金和有价证券+\(\sum_{t=1}^n\cfrac{FCF_t}{(1+r)^t}\)
      • =初始现金和有价证券+企业价值（EV）

2.3 无杠杆自由现金流

  • WACC，无杠杆自由现金流时的折现率 \[\begin{align} \because V&=\cfrac{V\cdot{r_{WACC}}}{r_{WACC}}=\cfrac{[\cfrac{E}{V}r_E+\cfrac{D}{V}r_D(1-T_C)]V}{r_{WACC}}\\ &=\cfrac{Er_E+Dr_D(1-T_C)}{r_{WACC}}\\ \because E&=\cfrac{(EBIT-Dr_D)(1-T_C)}{r_E}\\ \therefore V&=\cfrac{(EBIT-Dr_D)(1-T_C)+Dr_D(1-T_C)}{r_{WACC}}\\ &=\cfrac{EBIT(1-T_C)}{r_{WACC}} \end{align}\]

  • 无杠杆假设/全权益假设

    • 计算所得税时，假设公司没有负债，因此也没有利息，税后净利润=EBIT*（1-T）
    • 有杠杆假设下，税后净利润=（EBIT-利息）*（1-T）

  • 计算净营运资本变动时，只考虑OWC，不考虑现金及有价证券变动

2.3.1 无杠杆自由现金流的计算

  • 自上而下：
    • UFCF=EBIT（1-T）+折旧-CAPEX-ΔOWC
  • 自下而上：
    • UFCF=净利润+折旧+利息*（1-T）-CAPEX-ΔOWC
  • 粗略看，UFCF=经营活动产生的现金流-投资活动产生的现金流
    • 从经济含义上讲，公司的无杠杆自由现金流等于公司经营活动现金流入扣除投资需要的净流出
  • 比较现金流量表中经营活动产生的现金流与UFCF中OCF-∆OWC
    • 前者=净利润+折旧+利息-非现金流流动资产增加+非付息流动负债增加=净利润+折旧+利息-∆OWC
    • 后者=净利润+折旧+利息*（1-T)-经营性ΔOWC
    • 两者相差利息*T，即利息的税盾
    • 也就是说，如果用现金流量表算（现金流量表不使用无杠杆假设），最后还要减去利息税盾，才是UFCF

总结

  1. 合理预测FCF或UFCF
  2. 预测WACC
  3. 确定公司价值：公司价值=负债+股东权益=初始现金有价证券+EV
  4. 计算股东权益价值：股东权益价值=初始现金和有价证券+EV-负债

总结

  1. UFCF和CF（A）有啥区别：
    1. 前者用无杠杆假设
    2. 前者用OWC
  2. 自由现金流恒等式CF（A）=CF（S）+CF（B）是否用了无杠杆假设
    1. 没有用
  3. 如果用OWC计算，自由现金流恒等式是否成立
    1. 是
  4. 股权自由现金流是依据自由现金流恒等式算的，因此没有无杠杆假设
    1. 使用CF（A）-CF（B）来计算，算CF（A）时用OWC

Lecture 3 财务分析与财务建模

3.1 财务报表分析

3.1.1 水平分析

  • 把报表不同时期的同项数据进行对比，要注意科目之间的可比性

3.1.2 垂直分析

  • 把同一年的数据进行对比，比率分析

  • 五大类财务比率指标

  1. 盈利能力
    1. 销售回报率
      1. 毛利率
      2. 息税前利润率
      3. 营业利润率
      4. 税前利润率
      5. 净利率
        • =净利润/销售额
    2. 投资回报率
      1. 总资产回报率ROA
        • =净利润/总资产
      2. 净资产回报率ROE
        • =净利润/总权益
      3. 投入资本回报率ROIC
        • =息前税后利润/（债务+股东权益）
  2. 资产管理效率（天数就是365/周转率）
    1. 应收账款周转率/天数
      • =销售额/应收账款
    2. 存货周转率/天数
      • =销售成本/存货
    3. 应付账款周转率/天数
      • =销售成本/应付账款
    4. 现金周期
      • =应收账款周转天数+存货周转天数-应付账款周转天数
    5. 固定资产周转天数
    6. 总资产周转天数
      • =销售额/总资产
  3. 偿债能力（短期偿债能力：流动性指标）
    1. 流动比率
      • =流动资产/流动负债
    2. 速动比率
      • =（流动资产-存货）/流动负债
    3. 现金比率
      • =现金/流动负债
    4. 资产负债率
      • =（总资产-总权益）/总资产
    5. 负债权益比率
      • =总负债/总权益
    6. 利息覆盖倍数
      • =EBIT/利息
    7. 现金对利息的保障倍数
      • =EBITDA/利息
  4. 市场价值
    1. 市盈率
      • =每股价格/每股收益
    2. 市净率
      • =每股价格/每股净资产
    3. 股票市值
      • =每股价格*发行在外的股份数
    4. 企业价值
      • =股票市值+付息负债市场价值-现金
    5. 企业价值乘数
      • =EV/EBITDA

3.2 财务规划

  • 投资的新资产——资本预算决策
  • 流动性要求——营运资本政策决定
  • 财务杠杆程度——资本结构决策
  • 支付给股东的现金——股利政策决定

销售百分比法，四步： 1. 预测利润表 1. 确定销售增长率，预计销售收入 2. 确定利润表中各个项目按照增长率的结果 1. 折旧、利息费用等通常为常数 2. 股利根据股利政策 2. 预测资产负债表 1. 资产项目根据销售增长率变化 2. 应付账款根据销售增长率变化 3. 短期借款、长期负债和权益资本通常不随销售变化，根据资本结构决策 4. 留存收益变动根据股利政策 3. 预测额外资金需求 1. 算完之后，看看负债还差多少，计算EFN 2. \(EFN=\cfrac{资产}{销售额}\times\Delta销售额-\cfrac{自发增长负债}{销售额}\times\Delta销售额-销售利润率\times预计销售额\times(1-股利支付率)\) 3. 也可以用预测后的资产-负债和权益算，用来配平 4. 确定股利分配方案和外部融资方案 1. 更多短期应付票据 2. 更多长期负债 3. 更多普通股 4. 减少股利分配，增加留存收益

  • 前面假设外部融资需求的负债在年末借入，不影响利息

  • 如果负债在年初借入，会循环变动，用excel建模解决

  • 内部增长率：没有任何形式外部筹资，能达到的最大增长率

    • g=ROA*b，ROA为NI/A，b为留存比率

  • 可持续增长率：保持固定债权比例，没有任何形式外部权益融资，能达到最大增长率

    • g=ROE*b
    • 决定因素（杜邦分析）：
      • 销售利润率
      • 总资产周转率
      • 财务杠杆
      • 股利政策

3.3 财务建模

  • 将公司的各种信息按照价值创造的主线分类、整理和链接（核心业务）
  • 一般财务模型包括三个方面：
    • 历史分析
      • 历史经营绩效
      • 收入、费用、资产、负债等
      • 变动规律和趋势
      • 为预测提供依据
    • 财务预测
      • 考虑公司发展规划、战略、外部环境变动
      • 对未来收入增长、成本结构、资本性支出等做假设
      • 完成三表预测

      • 选择估值方法
      • 在预测基础上计算估值参数
      • 对公司价值进行评估
      • 分析重要参数对估值的影响
      • 敏感度分析（分析抗风险能力）
