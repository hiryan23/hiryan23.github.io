发展经济学-课程笔记

2023-02-16 2023-02-26 2022S发展经济学 发展经济学, 笔记 评论

本笔记整理自安泰经济与管理学院2022年春季学期课程ECON2602-02发展经济学，授课人是顾海英老师。

第一单元 绪论

发展经济学与经济学的关系

可以算宏观经济学，也可以算经济学流派

经济学的三个假定

经纪人假定、资源稀缺假定、保护私有财产假定

经济学的三项原理

利益最大化原理、供求原理、等价交换原理

经济学的三种方法

成本收益分析方法、均衡分析方法、帕累托标准

早期经济学思想对发展经济学形成和发展的影响

  1. 经济学开山鼻祖、亚当·斯密1776年出版的《国富论》：增长理论
  2. 李嘉图、马尔萨斯等把经济增长或经济停滞作为主要研究对象，创立古典增长理论。李嘉图：土地报酬递减规律；马尔萨斯：人口原理等
  3. 1870-二战，增长理论沉寂约100年。研究探讨短期静态的最优配置和均衡问题-边际革命，代表是马歇尔。静态的插曲：熊皮特创新理论。
  4. 马克思的《资本论：政治经济学批判》1867-1894，以剩余价值为中心；唯物史观；跨越多领域的分析
  5. 二战后，西方经济学家对增长和发展问题开始表现出浓厚的兴趣—发展经济学形成与发展的背景

思考

关注经济社会发展，开展调查研究；从现实矛盾中凝练出社会问题，进行学理研究；将研究成果转化为政策语言，为经济社会发展提供决策参考。

发展经济学的研究对象

发展经济学是专门研究发展中国家经济发展问题的经济学分支学科

  • 发展经济学是一门具有鲜明问题导向性质的学科
  • 以经济发展问题作为研究主题

发展中国家或不发达国家的经济发展问题
转型国家的转型发展问题
发达国家的发展问题

  • 经济发展本身是一个目标多维的动态过程

发展经济学以发展中国家的经济发展问题作为主要研究对象

  • 发展中国家指经济比较落后但正在发展中的国家

发展经济学作为一门实用性很强的学科，影响着发展中国家努力的方向和发展的进程

研究内容：

发展中国家经济发展问题可以概括为两个方面：

  1. 发展中国家经济相对于发达国家为什么落后或不发达？（探寻经济落后或不发达的根源）
  2. 发展中国家经济如何加快经济发展步伐来追赶当今发达国家？（实施何种战略和政策来加快经济发展）

研究发达国家经济发展曾经走过的发展道路和采取的发展政策，提供经验借鉴。

发展经济学研究的特点

  1. 着重探讨与长期变化有关的要素、制度、结构、战略与政策问题
    • 发展是一个长期的、动态的变化过程
  2. 着重结构转变分析
    • 经济发展过程本质上是一个结构转变过程——一个农业经济如何变为工业化经济的过程
  3. 强调非经济因素的影响分析
    • 如政治、社会、文化、生态、土地占有制度的变革与经济发展之间关系
  4. 更强调政府在经济发展中的重要作用
    • 发展中国家市场制度不完善，信息不灵敏，结构缺乏弹性，基础设施落后，再加上制度变革和结构转变以及资源调动等等，需要强大政府的推动

发展经济学学科的独特性

托达罗：

  • 尽管发展经济学常常采用标准经济学的原理和概念，但正在迅速形成特有分析和方法论特色
  • 发展经济学与资本主义和社会主义国家的经济学都毫不相似
  • 有着不同意识形态导向、多样化文化背景和非常复杂但又非常相似的经济问题，都要诉诸新思想和新方法

瑟尔沃：

  • 发展经济学家不仅创建了新的模型，而且依据环境而修正。可能应用于其他学科，大大地丰富了整个经济学
  • 由发展经济学家提出的概念和观点：低水平均衡陷阱、大推进、动态外部性、二元模型、循环累积因果关系、依附、增长极、人口与增长模型、乡-城人口流动、社会成本-收益分析、贫困化增长、结构性通货膨胀、两缺口分析、市场流失、寻租等

发展经济学的研究方法

  • 动态分析方法
  • 结构分析方法
  • 多维度的综合分析方法
  • 历史的比较分析方法

发展经济学的形成与发展

发展经济学形成于20世纪40-50年代

  • 背景：二战后，世界殖民体系开始土崩瓦解

亚非拉地区的殖民地或附属国相继脱离殖民统治，成为独立国家

新生的国家在经济上依然贫穷落后。如何加速经济发展、摆脱贫困成了发展中国家的主要目标

  • 发达国家的一些经济学家开始关注和研究发展中国家的经济问题

自20世纪40年代末以来，各种各样的发展理论相继问世，到20世纪50年代达到了高潮，形成了一门独立的学科——发展经济学

早期的经济发展思想 见上文

二战后发展经济学的发展背景

  • 世界殖民体系迅速瓦解，殖民地和附属国相继获得政治独立
    • 刚刚独立的国家，处于落后贫穷状态
      • 为了巩固政权，急于想把经济搞上去
      • 需要理论家当谋士，为经济发展出谋划策
      • 西方经济学家对发展中国家的问题产生兴趣
  • 苏美两大阵营展开竞赛
    • 欧美认为，共产主义的产生根源是贫困落后，为遏制共产主义，对穷国提供援助；建立援助计划，迫切需要研究发展中国家
  • 联合国和世行等机构编制国家统计资料
    • 提供了较为全面的资料和信息；这些资料在战前不可得到
  • 发达国家和发展中国家之间的联系越来越紧密
    • 国际贸易和国际投资发展很快，发展中国家的状况好坏直接影响发达国家

大量研究文献支撑学科形成

  • 科林·克拉克（Colin Clark）1940年出版《经济进步的条件》
  • 罗森斯坦-罗丹（P. Rosenstein-Rodan）1943年发表《东欧和东南欧工业化问题》
  • 斯塔利（E. Staley）1944年出版《世界经济发展》
  • 曼德尔鲍姆（K. Mandelbaum）1947年出版《落后地区的工业化》
  • ……
  • 中国学者张培刚教授在美国哈佛大学所作的博士论文《农业国工业化问题初探》在1949年由哈佛大学出版社出版，也是早期的重要文献
  • 英国经济学家哈罗德（R. Harrod）和美国经济学家多马（E. Domar）在1940年代分别提出了内容相似的增长理论，史称为哈罗德—多马增长模型
  • 在1950年代，对发展中国家经济发展问题研究形成了一股热潮，研究文献越来越多，仅10年左右时间，便形成了发展经济学这个具有独立研究对象和研究方法的新兴学科

发展经济学的演变

发展经济学经历了70多年的演变过程，大致上可分为三个阶段

  • 第一阶段：形成与繁荣时期——从20世纪40年代末到60年代中期
  • 第二阶段：新古典主义的复兴时期——从20世纪60年代中期到80年代初
  • 第三阶段：发展研究领域的拓展时期——从20世纪80年代中期到现在
第一阶段
  • 具有反新古典主义的倾向、注重内向发展战略、结构主义思路占主导地位、热衷建立宏大的理论体系
  • 具有“唯资本论”“唯工业化论”“唯计划论”的色彩
    • 强调资本积累。认为：资本稀缺是制约经济发展的主要因素
    • 强调工业化。认为：要摆脱贫穷，赶上发达国家，必须加快工业化
    • 强调计划化。认为：发展中国家市场体系不完善，价格机制不能很好发挥作用；政府如果不进行积极干预，有限社会资源不能有效使用
  • 三者相互联系，而非孤立：工业化是目标；资本积累是源泉；计划是手段
    • 没有物质资本积累就不可能实现工业化
    • 没有政府干预和计划安排，仅靠市场自发调节，不可能迅速进行资本积累，促进工业部门扩张，也不可能实现工业化
第二阶段
  • 把早期发展经济学家否定的东西重新加以肯定
  • 第一阶段的问题：
    1. 基本上把GDP增长作为发展首要目标，甚至是唯一目标。收入分配和贫困问题却趋恶化，失业问题更为严重
    2. 不仅不对农业进行投资，而且把农业部门作为榨取剩余的主要源泉
    3. 对市场机制的作用多持否定态度
    4. 长期实行内向型发展战略的国家碰到越来越多的困难
  • 针对问题：
    1. 出现一股重新解释发展的意义和重新确定发展目标的思潮
    2. 舒尔茨等抨击通过牺牲农业而实现工业化的做法，强调在工业化过程中应重视农业发展，通过对农业投入和农民人力资本投资来提高农业生产率，把传统农业转变为现代农业
    3. 提出“市场价格机制不仅能有效地配置资源并为经济增长提供刺激，而且作为一种有效管理工具，可用来完成政策目标，并避免直接控制下的低效率”的观点。
    4. 批评贸易条件恶化论、不平等交换和贸易悲观主义等观点，特别是抨击进口替代发展战略及相应的贸易保护政策
  • 关注贫困和不平等问题，重新确定发展目标；重视农业发展的重要性；重新强调和论证市场价格机制的作用；强调对外贸易在经济发展中的作用；重视国别和微观研究
第三阶段
  • 第二阶段的问题：
    • 新古典主义复兴对于推动发展经济学的发展虽起过积极的推动作用，但其存在的无制度背景、无时间维度、零交易成本等特点，难以分析和处理发展经济学中的许多问题；这种情况日益引起人们的关注和反思
  • 1980年代后，经济学在基本理论上的突破，如：新增长理论的兴起、可持续发展观的形成、新制度经济学的崛起和社会资本理论的渗透，尤其是中国特色的社会主义理论等，给发展经济学的发展注入了很大的活力
  • 第三阶段发展经济学的研究范围
    1. 把发展研究扩展到制度和政治、社会资本等其他非经济学领域
    2. 扩展到过去被认为是生态学和环境科学研究的领域
    3. 发展研究采用的仍然是新古典主义的分析方法
  • 强调制度及其变迁在经济发展中的重要性；新古典政治经济学方法在发展分析中广泛应用；现代增长理论的复兴以及与发展理论日益融合；资源、环境问题受到越来越多的重视；社会资本概念越来越多地渗透到发展研究中

发展思想的演进

  1. 发展的目标：国内生产总值（GDP）——实际人均国内生产总值——非货币指标（人类发展指数）——减缓贫困——权利与能力——自由——可持续发展……
  2. 宏观经济增长理论：哈罗德—多马分析——索罗增长源泉理论——新增长理论……
  3. 资本积累：有形资本——人力资本——知识资本——社会资本……
  4. 国家与市场：市场失灵——非市场失灵——新市场失灵——制度失灵……
  5. 政府干预：规划与计划——简洁的政府——政府与市场的互补……
  6. 政策改革：“因为贫穷而贫穷”——因“使价格失当”的不当政策而贫穷—“使所有政策失当”——”使制度失当”……

发展经济学的当代发展趋势

  1. 微观经济分析倾向越来越显著
    1. 分析主题微观化
      • 个人、家庭、社区、学校、企业
    2. 分析问题微观化
      • 教育、健康、营养、妇童、婚姻、家庭
  2. 发展的制度、文化和政治经济学分析思路更为突出
    • 公共产品、腐败、产权、公共治理、文化、社会信任、企业家精神……
  3. 在实证分析工具上取得了显著的进展
    • 反贫困评估、随机性方法……

发展经济学家

  • 第一代发展经济学家（1950-1975年）
  • 第二代发展经济学家（1975年-20世纪末）
  • 第三代发展经济学家的十大任务——杰拉尔德·迈耶（Gerald M. Meier）
    1. 超越收入增长：增长的格局与收入分配
    2. 创造就业
    3. 理解增长的源泉
    4. 制度的影响
    5. 追赶：技术和社会能力的作用
    6. 定义社会资本
    7. 金融制度的发展
    8. 全球化的含义
    9. 国家与市场的互补
    10. 政策制定和经济建议

发展经济学的学科地位

  1. 发展经济学研究发展中国家经济问题，但主要由发达国家的经济学家创立
    • 受西方正统经济学家影响，常运用正统经济学的基本原理和方法
    • 一些人认为，发展经济学不是一门独立学科，而只是把西方经济学的各个分支的基本原理在发展中国家的具体运用
    • 发展经济学已经形成一门独立的学科：有独特研究对象和方法——理论体系
  2. 学科体系不太完善
    • 发展中国家在经济、政治文化、地域规模上存在巨大差异
    • 建立一个对所有国家都适用的体系比较困难
    • 与宏观、微观经济学等学科相比，还不太完善
  3. 发展经济学本身也在发展之中
    • 根据发展中国家的历史经验教训而不断修正早期的观点，提出新的理论和政策

发展经济学在中国

发展经济学与中国经济学的发展

  • 1979年：中华外国经济学说研究会成立；范家骧举办发展经济学讲座

  • 1985-1989年：1985年谭崇台《发展经济学》（人民出版社）；1987、1988、1989年万晓光、陶文达、谭崇台三部《发展经济学》相继出版；《世界发展报告》中文版出版

  • 1991-1992年：1991年陶文达《中国社会主义经济发展概论》（辽宁人民出版社）；1992年张培刚《新发展经济学》……

  • 发展经济学在中国的使命

    • 统一基础的经济学原理不存在
    • 多种多样的应用经济学缺乏共同的理论基础……

  • 发展经济学作为经济学在中国的理论生长点发挥重要作用——中国发展经济学

  • 发展经济学的理论指导

    • 马克思历史唯物主义
      • 社会发展过程是生产力与生产关系、经济基础与上层建筑的矛盾运动过程

第二单元 发展中国家的特征

一个人的重大观点，包括经济观在内，来源并形成于生活实践和读书求知。研究经济学要懂得历史。——“发展经济学之父”张培刚

一、发达国家与不发达国家的形成

  • 经济世界（法国年鉴派史学大师布罗代尔Fernand Braudel）的组成要素：

    • 占据着一个特定的地理空间，有特定的界限
    • 有一个极或一个中心——有支配地位的都市
    • 有延缓的区域心脏——中心地区、中间地区、边缘地区（处于从属依附地位）

  • 现代区位理论先驱者屠能（Johann Heinrich von Thünen, 1783-1850）——出版3卷本《孤立国》（1826-1865）——提出“农业区位论”——屠能圈（Thünen Kreise）理论

  • 中世纪乃至古代——几个共存的经济世界天各一方、极少交往

  • 彼得大帝（1682-1725）之前——俄罗斯的经济世界自食其力，自我封闭

  • 18世纪末之前

    • 土耳其帝国的经济世界巨大
    • 查理五世或菲力普二世帝国则不是一个经济世界——从诞生起便被纳入由欧洲组成的经济巨网

西方世界的兴起过程

中世纪（贫困时期）——经济中农业占主导地位；经济组织形式：庄园；城市生活和贸易；决定交换条件的方法：习俗、惯例和法律

  • 13世纪：各种制度仍发挥作用
  • 14世纪后半叶：战争、瘟疫和饥荒，导致人口大量减少，有人居住的地方和耕地面积缩小
    • 庄园制（封建制）的本质：庄园主执行政府的职能——庄园是一个完整的政治和经济体系
    • 庄园制的根本特征：强制性劳动——刺激劳动的手段不仅仅有契约，还有政治和惩罚
    • 庄园的重要特点：自给自足

1500前：生活在世界各地的人们只有局部地接触和交往，没有人知道完整的世界究竟是什么样的

  • 中国：正处在明朝统治下，郑和船队的最后一次远航已经过去半个多世纪，对于本国疆域之外的世界并没有太多好奇
  • 阿拉伯和印度：其商人与欧、亚、非大陆都有贸易往来，但活动范围基本局限在印度洋上
  • 美洲：其文明与其他大洲之间横隔着两个大洋，相互之间毫无往来
  • 欧洲：相对独立和封闭，除了阿拉伯商人带来东方的讯息，欧洲人对于世界的认知，大多来自神话和幻想

15世纪（恢复时期）——造船技术取得进展；贸易和商业发展；航海家探险

  • 人口增长：城市发展；经济福利改善；农业和手工业生产方法改进；一些国家的庄园制农业变为个体所有制农业
  • 制度创新：形成稳定预期的法律体系；开始使用汇票；兴起保险市场；有系统的征税
    • 大规模的贸易发展使以血缘关系为基础的家庭商号向以非血缘关系为基础的企业发展：复式簿记；信任和忠诚；道德和宗教信仰体系
  • 政治因素：皇室当局与商人之间重商主义的合作关系；军事力量在几个皇家政府中（不是一个大帝国中）得到加强

18世纪（兴起时期）——持续增长所需要的制度前提在荷兰和英格兰建立起来，西方世界进入一个新经济时代的入口

经济世界中心的变迁

：约200年变迁一次

  • 1492年之前：欧洲——地中海——远东——中心威尼斯
  • 1492年哥伦布远行发现新大陆后：大西洋诸岛和沿岸——美洲内陆——印度、南洋群岛和中国——欧洲内部的中心由南向北迁移
  • 1500年前后：中心从威尼斯跳到安特卫普
  • 1500-1560年：中心回归地中海的热那亚
  • 1590-1610年：中心迁往阿姆斯特丹（跨两个世纪）
  • 1780-1815年：中心移往伦敦（跨两个世纪）
  • 1929年：中心定位于纽约（西方世界）
  • 9.11以来（2001年）……世界面临百年未有之大变局……
