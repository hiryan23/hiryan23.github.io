发展经济学

2023-02-16 课程笔记 发展经济学, 笔记 Comments

第一单元：绪论

发展经济学与经济学的关系

可以算宏观经济学，也可以算经济学流派

经济学的三个假定

经纪人假定、资源稀缺假定、保护私有财产假定

经济学的三项原理

利益最大化原理、供求原理、等价交换原理

经济学的三种方法

成本收益分析方法、均衡分析方法、帕累托标准

早期经济学思想对发展经济学形成和发展的影响

  1. 经济学开山鼻祖、亚当·斯密1776年出版的《国富论》：增长理论
  2. 李嘉图、马尔萨斯等把经济增长或经济停滞作为主要研究对象，创立古典增长理论。李嘉图：土地报酬递减规律；马尔萨斯：人口原理等
  3. 1870-二战，增长理论沉寂约100年。研究探讨短期静态的最优配置和均衡问题-边际革命，代表是马歇尔。静态的插曲：熊皮特创新理论。
  4. 马克思的《资本论：政治经济学批判》1867-1894，以剩余价值为中心；唯物史观；跨越多领域的分析
  5. 二战后，西方经济学家对增长和发展问题开始表现出浓厚的兴趣—发展经济学形成与发展的背景

思考

关注经济社会发展，开展调查研究；从现实矛盾中凝练出社会问题，进行学理研究；将研究成果转化为政策语言，为经济社会发展提供决策参考。

发展经济学的研究对象

发展经济学是专门研究发展中国家经济发展问题的经济学分支学科

  • 发展经济学是一门具有鲜明问题导向性质的学科
  • 以经济发展问题作为研究主题

发展中国家或不发达国家的经济发展问题
转型国家的转型发展问题
发达国家的发展问题

  • 经济发展本身是一个目标多维的动态过程

发展经济学以发展中国家的经济发展问题作为主要研究对象

  • 发展中国家指经济比较落后但正在发展中的国家

发展经济学作为一门实用性很强的学科，影响着发展中国家努力的方向和发展的进程

研究内容：

发展中国家经济发展问题可以概括为两个方面：

  1. 发展中国家经济相对于发达国家为什么落后或不发达？（探寻经济落后或不发达的根源）
  2. 发展中国家经济如何加快经济发展步伐来追赶当今发达国家？（实施何种战略和政策来加快经济发展）

研究发达国家经济发展曾经走过的发展道路和采取的发展政策，提供经验借鉴。

发展经济学研究的特点

  1. 着重探讨与长期变化有关的要素、制度、结构、战略与政策问题
    • 发展是一个长期的、动态的变化过程
  2. 着重结构转变分析
    • 经济发展过程本质上是一个结构转变过程——一个农业经济如何变为工业化经济的过程
  3. 强调非经济因素的影响分析
    • 如政治、社会、文化、生态、土地占有制度的变革与经济发展之间关系
  4. 更强调政府在经济发展中的重要作用
    • 发展中国家市场制度不完善，信息不灵敏，结构缺乏弹性，基础设施落后，再加上制度变革和结构转变以及资源调动等等，需要强大政府的推动

发展经济学学科的独特性

托达罗：

  • 尽管发展经济学常常采用标准经济学的原理和概念，但正在迅速形成特有分析和方法论特色
  • 发展经济学与资本主义和社会主义国家的经济学都毫不相似
  • 有着不同意识形态导向、多样化文化背景和非常复杂但又非常相似的经济问题，都要诉诸新思想和新方法

瑟尔沃：

  • 发展经济学家不仅创建了新的模型，而且依据环境而修正。可能应用于其他学科，大大地丰富了整个经济学
  • 由发展经济学家提出的概念和观点：低水平均衡陷阱、大推进、动态外部性、二元模型、循环累积因果关系、依附、增长极、人口与增长模型、乡-城人口流动、社会成本-收益分析、贫困化增长、结构性通货膨胀、两缺口分析、市场流失、寻租等

发展经济学的研究方法

  • 动态分析方法
  • 结构分析方法
  • 多维度的综合分析方法
  • 历史的比较分析方法
