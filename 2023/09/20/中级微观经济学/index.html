Kaldor-Hicks efficiency 卡尔多-希克斯有效

拍累托最优和功利主义有区别吗? - 知乎 (zhihu.com)

卡尔多改进，是指为了让输家接受再分配，再分配的赢家愿意付给输家的钱，高于输家愿意接受的心理价位；希克斯改进，是指为了让赢家放弃再分配，输家愿意付给赢家的钱，低于赢家的心理价位。

但是，这个效率有问题(Tibor Scitovsky, 1951) - Antisymmetric 反对称性：存在这样的情况，分配方案A是分配方案B的卡尔多-希克斯改进，但同时方案B也是分配方案A的卡尔多-希克斯改进。 - Intransitive 不可传递性：存在这样的情况，分配方案A是分配方案B的卡尔多-希克斯改进，分配方案B是分配方案C的卡尔多-希克斯改进，但是分配方案C又是分配方案A的卡尔多-希克斯改进。

Preference Relations

Binary relations

Let \(\succcurlyeq\) be a binary relation 二元关系 over \(X\) Let's call (i.e., def) this binary relation a preference relation 偏好关系 - If apple \(\succcurlyeq\) orange, we say "apple weakly preferred to orange" - Given \(\succcurlyeq\) , we can define \(\succ\) and \(\sim\) in the following way: - Indifference 无差异 - \(x\sim y\) : when \(x\succcurlyeq y\) and \(y \succcurlyeq x\) - Strict preference 严格偏好 - \(x \succ y\) : when \(x \succcurlyeq y\) and \(\lnot (y\succcurlyeq x)\)

Definition:

\(\succcurlyeq\) is reflexive 自反性 if \(x\succcurlyeq x\) for all \(x\in X\) \(\succcurlyeq\) is complete 完备性 if for all \(x,y\in X\), either \(x\succcurlyeq y\) or \(y\succcurlyeq x\) or both. ：完备性包含了自反性

  • Cal has preference \(\succcurlyeq = \emptyset\) (nothing is weakly preferred to anything)
    • neither reflexive nor complete
Definition:

\(\succcurlyeq\) is transitive 传递性 if \(x\succcurlyeq y\) and \(y\succcurlyeq z\) implies \(x\succcurlyeq z\)

  • Ellen has preference \(x\succcurlyeq y,y\succcurlyeq z, z\succ x\)
    • not transitive

Rationality

\(\succcurlyeq\) is rational if it is complete and transitive

If not, ...

  • That is, we understand people are different and may have different preference, but they "should" obey some basic properties
  • Hence, complete and transitive are often called axioms 公理

Utility

Utility Representation

Theorem 1a

Suppose \(X\) is finite. If \(\succcurlyeq\) is complete and transitive, then there exists a function \(U:X\rightarrow \mathbb{R}\) such that \(x\succcurlyeq y\Leftrightarrow U(x)\geq U(y)\).

That's called utility representation 效用表示

Theorem 1b

Suppose \(X\) is finite. If there exists a function \(U:X\rightarrow \mathbb{R}\) such that \(x\succcurlyeq y\Leftrightarrow U(x)\geq U(y)\), then \(\succcurlyeq\) is complete and transitive.

Utility maximization

Theorem 2

If \(U:X\rightarrow \mathbb R\) is a utility representation of \(\succcurlyeq\), then for all \(A\subseteq X\), \[\{x\in A:x\succcurlyeq y\ \forall\ y\in A\} \equiv \arg\max_{x\in A}U(x)\]

Utility representation theorem

  • Rational preference is sufficient (1a) and necessary (1b) for utility representation.
  • \(\Rightarrow\) The set of preference relations (people) that are rational is exactly the set that have utility representations.
  • \(\Longrightarrow\) If you believe people have complete and transitive preference relations, then you should accept the use of utility functions to represent these people.
  • \(\Longrightarrow\) If you further believe people maximize their preference, then you should accept a representation of utility maximization.

:不是所有理性的偏好都可以被效用函数表示，可数集总成立，不可数凸集要连续性 见下文

Ordinal vs Cardinal

Theorem 3

Fix \(\succcurlyeq\) . Suppose \(U:X\rightarrow \mathbb R\) represents \(\succcurlyeq\). \(V:X\rightarrow \mathbb R\) also represents \(\succcurlyeq\) if and only if \(V=f\circ U\) for some strictly increasing function \(f:\mathbb R\rightarrow \mathbb R\).

Why? - \(\succcurlyeq\) is a ranking of alternatives/bundles in \(X\) - Since \(U\) only conveys as much information as \(\succcurlyeq\), it cannot convey how much better or worse is \(x\) relative to \(y\).

  • Q: The same preference can be represented by different utility functions, what’s problematic about this?

Convex preferences

Definition

\(\succcurlyeq\) is - Convex - if \(x\succcurlyeq y\Rightarrow \alpha x+(1-\alpha)y\succcurlyeq y\ \forall\ \alpha\in[0,1]\) - Strictly convex 严格凸 - if \(x\succcurlyeq y\Rightarrow \alpha x+(1-\alpha)y\succ y\ \forall\ y\neq x\ \&\ \alpha\in(0,1)\)

Some kind of preference for balanced variety.

Theorem 4

If \(\succcurlyeq\) is rational + strictly convex and \(B\) is convex 凸集, then \(|B^*_\succcurlyeq|=1\).

Debreu's Theorem

A preference relation \(\succcurlyeq\)​​ that is complete, transitive, continuous, and defined over a convex subset of \(\mathbb R^n\), can be represented by a (continuous) utility function.

Lecture 3 Consumer Problem 1

Budget

  • If you have income \(m\) and prices are \((p_1,p_2)\), what can you afford?
  • Intuitively: affordable if \[x_1p_1+x_2p_2\leq m\]

Budget set

The set of all things that you can afford

  • Step 1: Find \((x_1,x_2)\) such that\[x_1p_1+x_2p_2=m\]
  • Step 2: Find \((x_1,x_2)\) such that\[x_1p_1+x_2p_2< m\]
  • Write: \[B(m,p_1,p_2)=\{(x_1,x_2)\in X|x_1p_1+x_2p_2\leq m\}\]
Questions
  • Does \(B(m, p_1,p_2)\) depend on utility?
    • No.
  • Does it depend on \(m,p_1,p_2\)?
    • Yes.
  • Does the fact that \(B\) is convex depend on \(m,p_1,p_2\)?
    • No.
  • Is it possible that \((m,p_1,p_2)\neq (m^\prime,p_1^\prime,p_2^\prime)\) but \(B(m,p_1,p_2)=B(m^\prime,p_1^\prime, p_2^\prime)\)?
    • Yes, by doubling for example.
N>2?

B is still convex (no matter the number of \(N\))

Distortions
  • Tax
    • Ex: When \(x_1\geq a\), you need to pay \(p_1+t\) for each unit of \(x_1\) above \(a\).

Relative price 相对价格

\[-\cfrac{p_1}{p_2}\] ... is the exchange rate between good 1 and good 2, without losing/gaining money, and is called the relative price

The consumer problem

The general problem

With utility function is \(U(x_1,\cdots, x_n)\), income \(m\), and prices \(p_1,\cdots,p_n\), what is your problem? \[\begin{align} \max U(x_1,\cdots,x_n)\\ s.t.\ x_1p_1+\cdots+x_np_n&\leq m\\ x_1\geq 0,\cdots,x_n&\geq 0 \end{align}\]

Indifference Curves

Brute force

Find the point \((x_1,x_2)\) in the budget set that achieves the highest \(U(x_1,x_2)\).

Indifference curves

Given a \(U(x_1,x_2)\) , you can draw the indifference curves 无差异曲线 (also called iso-utility curves, because utility stays the same)

An intuitive approach

If \(x^*\) is optimal bundle 最优消费束, what must be true/ what can we learn? - Necessary conditions 必要条件:
1. Just affordability 2. Marginal rate of substitution (MRS) condition

Necessary condition 1: Just affordable
Theorem 1

If \(U(\pmb x)\) is strictly increasing in at least one \(x_i\) , then \(\pmb x^*\) satisfies \[\pmb x^*\pmb p=m\]Proof: If not, spend the extra money on \(x_i\) for a strictly higher utility. It does not depend on the curves!

Necessary condition 2: MRS Condition
Marginal utility
  • Define Marginal Utility 边际效用: \[MU_i(\pmb x):=\cfrac{\partial U}{\partial x_i}(\pmb z)\Bigg |_{\pmb z=\pmb x}\]
Theorem 2a

If \(\pmb x^*\) satisfies \(x_i^*> 0\), then for all \(j\) \[\cfrac{MU_i(\pmb x^*)}{p_i}\geq\cfrac{MU_j(\pmb x^*)}{p_j}\] Proof: If \(LHS< RHS\), decrease consumption of \(x_i\) by \(\varepsilon > 0\) and increase consumption of \(x_j\) by \(\varepsilon \cfrac{p_i}{p_j}\). As \(\varepsilon \rightarrow 0\), gain in utility is arbitrarily close to \(\varepsilon\cfrac{p_i}{p_j}MU_j(\pmb x^*)\), loss in utility is arbitrarily close to \(\varepsilon MU_i(\pmb x^*\)). Since \(\cfrac{MU_I(\pmb x^*)}{p_i}< \cfrac{MU_j(\pmb x^*)}{p_j}\), \(Gain> Loss\). So \(\pmb x^*\) is not optimal.

Theorem 2b

If \(\pmb x^*\) satisfies \(x_i^*> 0\) and \(x_j^*> 0\) then \[\cfrac{MU_i(\pmb x^*)}{p_i}=\cfrac{MU_j(\pmb x^*)}{p_j}\] 边际效益越大，价格就越贵（？

MRS Condition

  • A relationship, under optimality, between preference and budget

  • Define the Marginal Rate of Substitution 边际替代率: \[MRS_{ji}(\pmb x):=-\cfrac{MU_i(\pmb x)}{MU_j(\pmb x)}\]

  • Then, \(\cfrac{MU_i(\pmb x^*)}{p_i}=\cfrac{MU_j(\pmb x^*)}{p_j}\Longleftrightarrow MRS_{ji}(\pmb x^*):=-\cfrac{p_i}{p_j}\)

  • MRS is the slope 斜率 of the indifference curve at point \((x_1,x_2)\)

  • It also is the marginal rate of substitution

  • If we bring in the budget set, necessary for optimality of \((\bar x_1,\bar x_2)\): \[MRS_{21}(\bar x_1,\bar x_2)=-\cfrac{p_1}{p_2}\]

Example

Theorem 3

If \(U,\tilde U\) represent the same preference (and technical conditions about differentiability 可导性), then for all \(i,j\) and \(\pmb x\in X\), \[\cfrac{MU_i(\pmb x)}{MU_j(\pmb x)}=\cfrac{M\tilde U_i(\pmb x)}{M\tilde U_j(\pmb x)}\]

Lecture 4 Consumer Problem 2

Necessary conditions

  • Theorem (1): If \(U(\pmb x)\) is strictly increasing in at least one \(x_i\) , then \(x^*\)satisfies \[x^*p=m\]

  • Theorem (2): If \(x^*\) satisfies \(x_i>0\) and \(x_j>0\) then \[\cfrac{MU_i(\pmb x^*)}{p_i}=\cfrac{MU_j(\pmb x^*)}{p_j}\]

  • The problem \[\begin{align} \max_x U(\pmb x)\\ s.t.\ \pmb x\pmb p&\leq m\\ x_i&\geq 0\ \forall i \end{align}\]

  • \(n\) unknowns and \(n\) equations, is it Done?

    • NO! (Because it is only necessary conditions)

Insufficient

An example
  • \(U(x_1,x_2)=x_1+\alpha x_2,\ \alpha>0\)
  • \(\cfrac{MU_1}{p_1}=\cfrac{1}{p_1},\ \cfrac{MU_2}{p_2}=\cfrac{\alpha}{p_2}\), \(p_1,\ p_2,\ \alpha\) are given
  1. \(\cfrac{1}{p_1}>\cfrac{\alpha}{p_2}\Rightarrow x_1^*=\cfrac{m}{p_1},\ x_2^*=0\)
  2. \(\cfrac{1}{p_1}<\cfrac{\alpha}{p_2}\Rightarrow x_1^*=0,\ x_2^*=\cfrac{m}{p_2}\)
  3. \(\cfrac{1}{p_1}=\cfrac{\alpha}{p_2}\Rightarrow x_1^*=\cfrac{m\beta}{p_1},\ x_2^*=\cfrac{m(1-\beta)}{p_2}\)
Discussion on Theorem (1) & (2)
  • \(x_1p_1+x_2p_2=m\)
    • by itself is clearly not sufficient. You spent all your money, but it doesn’t mean you spent them optimally
  • \(MRS=-price\ ratio\)
    • by itself is also not sufficient. You can achieve tangency (切线) without being on the budget line, so you “left money behind”, and this is not optimal
(2) holds but (1) fails
  • EX: \(U\) is a satiation preference
    • This guy doesn’t like as many kids or cats as possible.
    • Theorem (1) does not apply.
    • Solution is \(x_1^*,x_2^*\), not on the budget line. ##### (1) holds but (2) fails
  • But even if \(U\) is strictly increasing, (2) is after all a first order condition 一阶条件

Well-behaved Preferences 良性偏好

Definitions

To make (1)+(2) sufficient, we focus on a “special” class of \(U\)’s: 1. \(\succcurlyeq\) is Monotonic 单调: \(x_i\geq y_i\) for all \(i\) & \(x_i>y_i\) for some \(i\) \(\Rightarrow\) \(U(\pmb x)>U(\pmb y)\) 2. \(\succcurlyeq\) is Convex 凸: for all \(\alpha\in(0,1),\ U(\alpha\pmb x+(1-\alpha)\pmb y)\geq\min\{U(\pmb x),U(\pmb y)\}\) 3. ≽ is Well-behaved: Monotonic and Convex 4. …Strictly Convex: \(U(\alpha\pmb x+(1-\alpha)\pmb y)>\min\{U(\pmb x),U(\pmb y)\}\) for all \(\alpha\in(0,1), \pmb x\neq\pmb y\)

Monotonicity
  • Utility function that is strictly increasing in all goods
  • If \(U\) is differentiable: \(MU_i(\pmb x)>0\) for all \(i\)
Convex
  • Strictly Convex
  • Convex but not Strictly Convex

Sufficiency

Theorem 3

Suppose \(U\) represents a well-behaved preference. If \(\pmb x\) satisfies (1)+(2) and \(x_i>0\) for all \(i\) then it is a solution. - Proof sketch: Use separating hyperplane theorem 超平面分离定理 Warning: This \(\pmb x\) need not exist, even if there is a solution.

Theorem 4

Suppose \(U\) represents a well-behaved strictly convex preference. If \(x\) satisfies (1)+(2) and \(x_i\geq0\) for all \(i\) then it is the unique solution.

一个傻逼函数\(y=(x_1x_2)^k\)

  • Hessian矩阵 \[\begin{bmatrix} k(k-1)x_1^{k-2}x_2^k & k^2x_1^{k-1}x_2^{k-1}\\ k^2x_1^{k-1}x_2^{k-1} & k(k-1)x_1^kx_2^{k-2}\\ \end{bmatrix}\]

  • 凹性条件：Hessian矩阵（半）负定 \[\begin{align} k(k-1)x_1^{k-2}x_2^k<0\\ \\ \begin{vmatrix} k(k-1)x_1^{k-2}x_2^k & k^2x_1^{k-1}x_2^{k-1}\\ k^2x_1^{k-1}x_2^{k-1} & k(k-1)x_1^kx_2^{k-2}\\ \end{vmatrix}>(\geq)0 \end{align}\Rightarrow \begin{cases} 0<k<1\ \text{or}\ k>1\\ k<(\leq)0.5 \end{cases} \]

  • 拟凹性条件? \[\begin{align} f_{11}f_2^2-2f_{12}f_1f_2+f_{22}f_1^2<0\\ \\ LHS\Leftrightarrow 2k^3x_1^{3k-2}x_2^{3k-2}(-1) \end{align}\]

  • 所以 \[y是\begin{cases} k<0,\ \ \ \ &非拟凹函数，凸函数\\ k=0,\ \ \ \ &\equiv1\\ 0<k<(\leq)0.5,\ \ \ \ &拟凹函数，凹函数\\ k>0.5,\ \ \ \ &拟凹函数，不是凹函数，也不是凸函数 \end{cases}\]

Under monotonic preference:

  • Method 2: Plot and check that indifference curves are convex (slope/\(MRS_{ij}\) along an indifference curve is non-decreasing in \(x_j\))

  • Method 3: Sufficient condition: \(MU_{ii}\leq 0\) and \(MU_{ij}\geq 0\)

    • \(MU_{ii}\leq 0\): diminishing marginal utility
      • greater willingness to give up \(x_i\) for other goods when \(x_i\) is high
    • \(MU_{ij}\geq 0\): complementarity
      • greater willingness to give up other goods for \(x_j\) when other goods are high

Cautionary Example 1

  • \(U(x_1,x_2)=x_1+x_2^2\)
  • \(MRS_{21}(x_1,x_2)=-\cfrac{MU_1}{MU_2}=-\cfrac{1}{2x_2}\)
  • Preference is not convex!

Cautionary Example 2

  • \(U(x_1,x_2)=x_1^2+x_1x_2+x_2\)
  • Indifference curve at \(U=a\) is \(x_2=\cfrac{a-x_1^2}{1+x_1}\).
    • \(\cfrac{\text{d} x_2}{\text{d} x_1}=MRS_{21}(x_1,a)=-\cfrac{a+x_1^2+2x_1}{(1+x_1)^2}\)
    • \(\cfrac{\text{d}^2 x_2}{\text{d} x_1^2}=\cfrac{2(a-1)}{(1+x_1)^3}\)
  • When \(a>1\), this indifference curve is convex, but when \(a<1\), it is concave.

Lagrange

\[\begin{align} &\max_{x_1,\cdots,x_n}U(x_1,\cdots,x_n)\\ &s.t.\ x_1p_1+\cdots+x_np_n=m \end{align}\] Turns to \[\begin{align} &\max_{x_1,\cdots,x_n,\lambda}U(x_1,\cdots,x_n)-\lambda f(x_1,\cdots,x_n)\\ &s.t.\ f(x_1,\cdots,x_n)=x_1p_1+\cdots+x_np_n-m \end{align}\] 1. \(\lambda\) is sometimes called the shadow price. It represents the marginal utility of income at optimal consumption (as you will see). 2. Therefore, this method exploits the a binding budget constraint to bring the trade-off between choice variables \(x_1,\cdots,x_n\) into the maximization problem by linking them to \(\lambda\).

  • FOCs:
    • \(x_1p_1+\cdots+x_np_n=m\)
    • \(\cfrac{MU_i}{p_i}=\lambda=\cfrac{MU_j}{p_j}\Rightarrow MRS_{ji}=-\cfrac{p_i}{p_j}\) for all \(i,j\)

Karush-Kuhn-Tucker (KKT条件)

A method to solve optimization problem for which constraints hold, but need not bind. ### Some rule of thumb Remember that necessary conditions may not be sufficient. 1. If preference is well-behaved, check if any \(x^*\geq 0\) satisfies (1)+(2), they are the solutions. If not, check corner bundles. 2. If preference is monotonic, check if any \(x^*\geq 0\) satisfies (1)+(2), compare utility among them and with corner bundles. If not, check corner bundles. 3. Useful fact: If \(MU_i(\pmb x)\rightarrow \infty\) when \(x_i\rightarrow 0\) (Inada condition 稻田条件), then there is no solution such that \(x_i=0\) (no corner solutions 边角解).

Common Examples

Must know

1. Cobb-Douglas Preference 柯布-道格拉斯

\[U(x_1,x_2)=x_1^cx_2^d\ \text{where}\ c,d\in(0,1)\] - Monotonic - Obvious - Convex - \(MU_1(\pmb x)=cx_1^{c-1}x_2^d\) decreasing in \(x_1\) and increasing in \(x_2\). - \(MU_2(\pmb x)=dx_1^1x_2^{d-1}\) decreasing in \(x_2\) and increasing in \(x_1\). - Inada - \(MU_1(\pmb x)=cx_1^{c-1}x_2^d\rightarrow\infty\) when \(x_1\rightarrow 0\). - \(MU_2(\pmb x)=dx_1^1x_2^{d-1}\rightarrow\infty\) when \(x_2\rightarrow0\). - No corner solution!

  • In this case, we go ahead with (1)+(2): \[MRS_{21}(\pmb x)=-\cfrac{MU_1(\pmb x)}{MU_2(\pmb x)}=-\cfrac{cx_1^{c-1}x_2^d}{dx_1^cx_2^{d-1}}=-\cfrac{c}{d}\cfrac{x_2}{x_1}\]\[ \left. \begin{align} x_1p_1+x_2p_2&\equiv m\\ MRS_{21}(\pmb x)&\equiv -\cfrac{p_1}{p_2}\Leftrightarrow\cfrac{c}{d}\cfrac{x_1}{x_2}=\cfrac{p_1}{p_2} \end{align} \right\} \Longrightarrow x^*=\left(\cfrac{c}{c+d}\cfrac{m}{p_1},\cfrac{d}{c+d}\cfrac{m}{p_2}\right) \] \(\Rightarrow\) Spend \(\cfrac{c}{c+d}\) fraction of income on \(x_1\), \(\cfrac{d}{c+d}\) fraction on \(x_2\).

2. Quasi-linear Preferences 拟线性

\[U(x_1,x_2)=\ln x_1+x_2\] - This is often used to characterize utility for money (as good 2) - Preference has feature: \((x_1,x_2)\succcurlyeq(y_1,y_2)\Rightarrow (x_1,x_2+z)\succcurlyeq(y_1,y_2+z)\) - More on this when we get to Chapter Consumer Surplus

  • Monotonic, strictly convex, but not Inada (in good 2)

\[MRS_{21}(\pmb x)=-\cfrac{MU_1(\pmb x)}{MU_2(\pmb x)}=-\cfrac{1/x_1}{1}=-\cfrac{1}{x_1}\] \[x^*= \begin{cases} \left(\cfrac{m}{p_1},0\right)\ \ \ \ &\text{if}\ m\leq p_2\\ \left(\cfrac{p_2}{p_1},\cfrac{m}{p_2}-1\right)\ \ \ \ &\text{if}\ m>p_2 \end{cases}\]

3. Perfect Substitutes 完全替代

\[U(x_1,x_2)=x_1+\alpha x_2,\ \alpha>0\] \[\cfrac{MU_1}{p_1}=\cfrac{1}{p_1}\ \ \ \ \cfrac{MU_2}{p_2}=\cfrac{\alpha}{p_2}\] \[\begin{align} \cfrac{1}{p_1}>\cfrac{\alpha}{p_2}&\Rightarrow x^*=\left(\cfrac{m}{p_1},0\right)\\ \cfrac{1}{p_1}<\cfrac{\alpha}{p_2}&\Rightarrow x^*=\left(0,\cfrac{m}{p_2}\right)\\ \cfrac{1}{p_1}=>=\cfrac{\alpha}{p_2}&\Rightarrow x^*=\left(\cfrac{m\beta}{p_1},\cfrac{m(1-\beta}{p_2}\right)\\ \end{align}\]

Perfect Complements 完全互补

\[U(x_1,x_2)=\min\{x_1,2x_2\}\] - Budget: \(x_1^*p_1+x_2^*p_2=m\) - Necessary: \(x_1^*=2x_2^*\) \[\begin{align} x_1^*&=2\left(\cfrac{m}{p_1+2p_2}\right)\\ x_2^*&=\left(\cfrac{m}{p_1+2p_2}\right) \end{align}\]

Lecture 5 Revealed Preference 1

Preference vs Choice

Suppose we observe, from one individual, a dataset of choice sets and choices - When are they consistent with the maximization of some preference？ - Throughout, we focus on the case without indifference

Primitive

  • "\(c(B)=x\)" means choosing \(x\) from the choice set \(B\subset X\)
  • "\(c(\{x,y,z\})=x\)" means choosing \(x\) from \(x,y,z\)
  • Each is a decision
  • A dataset is a collection of decisions made by an individual

Example #3

A dataset: \(c(\{x,y\})=x,c(\{y,z\})=y,c(\{x,z\})=z\) - Q1: Could he be maximizing a preference? - Yes. - Q2: Could he be maximizing a rational preference? - No.

The emphasis

  • We start with a dataset of choices (of one individual)
  • And then ask if it could come from a rational preference

Choices are real, preferences are theories (stories) that attempt to explain something real. - Some choices could not have come from a rational preference - So choices that could come from a rational preference will need to satisfy some properties

WARP and SARP 显示偏好弱（强）公理

Primitive

  • \(X\): a set of alternatives
  • \(A\): a choice set, nonempty subset of \(X\)
  • \(\mathcal A\): a collection of choice sets
  • \(c: \mathcal A\rightarrow X\) such that \(c(A)\in A\): a choice function / dataset

Model

Definition: \(c\) is rationalized 理性化 by a (strict) preference relation \(\succ\) if \[ c(A)\equiv\{x\in A: x\succ y\ for\ all\ y\in A\backslash\{x\}\}\ for\ all\ A\] - Q1: Is there a \(c\) that can be rationalized by a \(\succ\), but not by any rational \(\succ\)? - \(c(\{x,y,z_1\})=x,c(\{x,y,z_2\})=y\). It's not rational. - Q2: Is there a \(c\) that cannot be rationalized by any \(\succ\)? - \(c(\{x,y,z\})=x,c(\{x,y\})=y\)

可理性化的定义

We say \(C\) is rationalizable if it satisfies:
(Condition \(\alpha\)​​​): If \(x\in A\subset B\)​​ and \(x\in C(B)\)​​, then \(x\in C(A)\)​​​.
(Condition \(\beta\)): If \(x,y\in C(A)\), \(A\subset B\) and \(y\in C(B)\), then \(x\in C(B)\).

  • Sen 对于 Condition \(\alpha\) 的解释: 如果某场世界级比赛的冠军是一个巴基斯坦人，那么这个人也是巴基斯坦的冠军。
  • Sen 对于 Condition \(\beta\) 的解释: 如果某场世界级比赛的冠军是（几个）巴基斯坦人，那么所有这场比赛中的巴基斯坦人都是世界冠军。

Preference Revelation

Definition: If \(c(A)=x\) and \(y\in A\backslash\{x\}\), let \(xR_Dy\).

  • "\(x\) is directly revealed preferred 直接显示偏好 to \(y\)"
  • \(R_D\) is derived from choice data
  • Therefore, it does not (yet) guarantee a rational preference
Theorem 1

If \(c\) is rationalized by a rational \(\succ\) , then \(xR_Dy\Rightarrow\neg(yR_Dx)\).

Weak Axiom of Revealed Preferences (WARP) 显示偏好弱公理

\[\text{If}\ xR_Dy,\ \text{then not}\ yR_Dx\] - A property of a dataset - If a dataset violates WARP, then it cannot come from the maximization of a rational preference.

但是，由于有Example #3这种偏好，满足WARP，但仍不是理性的。所以还需要更多。

Indirect Preference Revelation

Definition: \[\text{If}\ xR_Dx\ \text{and}\ zR_Dy,\ \text{then let}\ xR_Iy\]

  • "\(x\) indirectly is revealed preferred 间接显示偏好 to \(y\)"
  • We (may) never see \(x\) chosen in the presence of \(y\)
    • But we see \(x\) chosen over \(z\) and \(z\) chosen over \(y\)
    • So we conclude, indirectly, that the 𝑥 is preferred to 𝑦
Theorem 2

If \(c\) is rationalized by a rational \(\succ\), then \(xR_Dy\) or \(xR_Iy\Rightarrow\neg(yR_Dx)\) and \(\neg(yR_Ix)\).

Strong Axiom of Revealed Preferences (SARP) 显示偏好强公理

\[\text{If}\ xR_Dy\ \text{or}\ xR_Iy,\ \text{then neither}\ yR_Dx\ \text{nor}\ yR_Ix\] - Again, it is a property of a dataset

Back to Example #3
  1. If a guy has \(c(\{x,y\})=x,c(\{y,z\})=y,c(\{x,z\})=z\)
  2. This gives \(xR_Dy,yR_Dz\), and \(zR_Dx\)
  3. And therefore WARP is satisfied
  4. But since \(xR_Dy\) and \(yR_Dz\), we set \(xR_Iz\)
  5. But then \(zR_Ix\) and \(xR_Iz\), so SARP is falsified
  • Q1: Can a dataset satisfy WARP but not SARP?
    • Yes.
  • Q2: Can a dataset satisfy SARP but not WARP?
    • No.
  • Found two necessary conditions 必要条件
  • A dataset come from rational preference must satisfies SARP & WARP

Representation Theorem 表示定理

Theorem 3

An exhaustive dataset of choices can be rationalized by a rational preference relation if and only if it satisfies WARP. exhaustive: Every (nonempty) \(A\subset X\) is in \(\mathcal A\)

  • "Only if" part is Theorem 1
  • Proof (If)
    1. Construction: Given \(c\), let \(x\succ y\) if there exists \(A\) such that \(c(A)=x\)
    2. Show complete: From \(A=\{x\}\) , we have \(x\succ x\). From \(A=\{x,y\}\), we have \(x\succ y\) or \(y\succ x\).
    3. Show transitive: Knowing \(\succ\) is complete, suppose for contradiction \(x\succ y\), \(y\succ z\), and \(z\succ x\). Consider \(A^*=\{x,y,z\}\), which is in \(\mathcal A\) since it is exhaustive. Suppose WLOG \(c(A^*)=x\), then \(xR_Dz\), which contradicts WARP.
    4. Show \(\succ\) rationalizes \(c\): Consider \(A\in\mathcal A\). By construction, \(c(A)\succ y\) for all \(y\in A\backslash\{c(A)\}\), so we just need to make sure there is no \(y\in A\backslash\{c(A)\}\) such that \(y\succ z\) for all \(z\in A\backslash\{y\}\) . Suppose for contradiction there is, then in particular we have \(y\succ c(A)\) , which means there is a choice set \(B\) such that \(c(B)=y\) and \(c(A)\in B\). But this implies \(yR_Dc(A)\) , which contradicts WARP.
  • If an exhaustive dataset of choices satisfies WARP, then it is as-if the individual makes decision by maximizing a rational preference.
  • WARP is sufficient and necessary for rationalization

Theorem 4

A dataset of choices can be rationalized by a rational preference relation if and only if it satisfies SARP.

  • "Only if" part is Theorem 2
  • The proof is more involved.
    1. We build \(R_D\) and \(R_I\) using the dataset of choices.
    2. We write \(xR_Cy\) if either \(xR_Dy\) or \(xR_Iy\) or both.
    3. Note that at this point, \(R_C\) is neither guaranteed complete nor guaranteed transitive.
    4. By SARP, \(R_C\) is acyclic (if \(x_1R_Cx_2,x_2R_Cx_3\cdots,x_{n-1}R_Cx_n\) where \(x_i\neq x_j\) , then not \(x_nR_Cx_1\)).
    5. Then, it is well-known that a transitive closure of \(R_C\) exists, denoted by \(R_C^\prime\) .
    6. Now we arbitrarily "join" the chains in \(R_C^\prime\) and let \(\succ\) be the output, which is complete and transitive.
    7. Finally, we need to show that \(\succ\) rationalizes the dataset of choices (for example, by arguments of contradictions).

Comments

  • Theorem 3 (and 4) is a watered-down version of the “2nd Theorem” in a field in economics called decision theory
  • It captures rationalizable choice if and only if rational preference.
  • The “1st Theorem” is “utility representation if and only if rational preference”, which you saw in Lecture 2
  • Together, they link choice and utility, fact and intuitions, falsifiable and thought-exercise, reality and models

Back to the Consumer Problem

Lecture 5 Theorem 4

A dataset of choices \(c:\mathcal A\rightarrow X\) can be rationalized by a rational preference relation if and only if it satisfies SARP.

  • This theorem works for arbitrary choice sets and arbitrary choices, including the consumer problem
  • But, we cannot observe all choice sets in a consumer problem.
  • So how can we define \(R_D\) and \(R_I\) ?

Inferred Budget

  • When the consumer chooses \(\pmb x\) at \(\pmb p\), the set \(\{\pmb z:\pmb z\pmb p\leq\pmb x\pmb p\}\) contains affordable bundles
  • So, we conclude \(\pmb xR_D\pmb x^\prime\) for all \(\pmb x^\prime\in\{\pmb z:\pmb z\pmb p\leq\pmb x\pmb p\}\backslash\{\pmb x\}\)

Assumptions

Additional assumptions can help us better infer preference, especially suitable when these assumptions make sense.

With loss, we focus on underlying preferences that are… 1. Strictly convex : for unique solution 2. Monotonic : so that choices must be just affordable

Example choice gives \(R_D\)

1. \(\pmb xR_D\pmb y\) 2. \(\pmb xR_D\pmb z\) 3. Neither \(\pmb xR_D\pmb a\) nor \(\pmb aR_D\pmb x\) 4. \(\pmb bR_D\pmb x\)

Example: from \(R_D\) to \(R_I\)

\(\pmb x^* R_D\pmb y^*\) and \(\pmb y^*R_D\pmb z\), so \(\pmb x^*R_I\pmb z\)

"Qualifying dataset"

Given a finite collection of choice-price pairs \((\pmb x,\pmb p)\) , construct a qualifying dataset: 1. Add to every data point the income \(m:=\pmb x\pmb p\), so that each data point becomes \((\pmb x,\pmb p,m)\) 2. Check no two datapoints \((\pmb x,\pmb p,m)\) and \((\pmb x^\prime,\pmb p^\prime, m^\prime)\) are such that \(\pmb p=\pmb p^\prime\) and \(m=m^\prime\) but \(\pmb x\neq\pmb x^\prime\)

Definitions: 1. Given \(U,\pmb x^*(\pmb p,m)\) denotes the solution to \(\displaystyle\max_{\pmb x} U(\pmb x)\) subject to \(\pmb x\pmb p\leq m\). 2. A \(WB-U\) is a utility function that represents well-behaved preference.

Theorem 1

Given a qualifying dataset, the following are equivalent: 1. The dataset satisfies SARP. 2. There exists a \(U\) such that if \((\pmb x,\pmb p,m)\) is in the dataset, then \(\pmb x=\pmb x^*(\pmb p,m)\). 3. There exists a \(WB-U\) such that if \((\pmb x,\pmb p,m)\) is in the dataset, then \(\pmb x=\pmb x^*(\pmb p,m)\).

Utility maximization of budget constrained individuals gives \[x_1^*(p_1,\bar p_2,\cdots,\bar p_N,\bar m)=x_1\]

(Ordinary) Demand Functions

Ordinary demand

\[x_1^*(p_1,\bar p_2,\cdots,\bar p_N,\bar m)=x_1\] Call it an ordinary (Marshallian) demand function

Cobb-Douglas Example

Consumer maximizes \(U(x_1,x_2)=x_1^ax_2^b\) subject to budget constraint \(\pmb x\pmb p\leq m\) - Optimal consumption (Lecture 4): \[\pmb x^*(p_1,p_2,m)=\left(\cfrac{a}{(a+b)}\cfrac{m}{p_1},\cfrac{b}{(a+b)}\cfrac{m}{p_2}\right)\] - Ordinary demand functions: \[x_1^*(p_1,\bar p_2,\bar m)=\cfrac{a}{(a+b)}\cfrac{\bar m}{p_1},\ \ \ \ x_2^*(\bar p_1,p_2,\bar m)=\cfrac{b}{(a+b)}\cfrac{\bar m}{p_2}\]

Own-price Changes

A general approach

  • “Changes of (optimal) consumption of \(x_i\) when \(p_i\) changes, holding \(m\) and other prices constant.”

Price offer curve and Demand curve

Example 1

Cobb-Douglas: \[\pmb x^*(p_1,p_2,m)=\left(\cfrac{a}{(a+b)}\cfrac{m}{p_1},\cfrac{b}{(a+b)}\cfrac{m}{p_2}\right)\]

Example 2

Perfect complements: \(U(x_1,x_2)=\min\{x_1,x_2\}\) \[\pmb x^*(p_1,p_2,m)=\left(\cfrac{m}{p_2+p_1},\cfrac{m}{p_2+p_1}\right)\]

Example 3

Perfect substitutes: \(U(x_1,x_2)=x_1+x_2\) \[\pmb x^*(p_1,p_2,m)= \begin{cases} \left(0,\cfrac{m}{p_2}\right)\ \ \ \ &p_1>p_2\\ \left(\cfrac{\beta m}{p_1},\cfrac{(1-\beta) m}{p_2}\right)\ \ \ \ &p_1=p_2\\ \left(\cfrac{m}{p_1},0\right)\ \ \ \ &p_1<p_2 \end{cases} \]

Cross-price Changes

  • If \(p_j\uparrow\) leads to \(x_i\uparrow\), we call \(i\) a gross substitute 总替代品 for \(j\)
  • If \(p_j\uparrow\) leads to \(x_i\downarrow\), we call \(i\) a gross complement 总互补品 for \(j\)

Income Changes

A general approach

  • "Changes of (optimal) consumption of \(x_i\) when \(m\) changes, holding prices constant."

Income offer curve and Engel Curve

Example 1

Cobb-Douglas \[\pmb x^*(p_1,p_2,m)=\left(\cfrac{a}{(a+b)}\cfrac{m}{p_1},\cfrac{b}{(a+b)}\cfrac{m}{p_2}\right)\] - Engel curves: \[\begin{align} m=\cfrac{(a+b)p_1}{a}x_1\\ m=\cfrac{(a+b)p_2}{b}x_2 \end{align}\]

Types of Commodity

Income Effects

Definitions: when \(m\) increases… 1. Normal good 正常商品 if \(x_i^*(\pmb p,m)\) always increases - Engel curve is positively sloped 2. Inferior good 低档商品 if \(x_i^*(\pmb p,m)\) sometimes decreases - Engel curve is sometimes negatively sloped

Normal vs Inferior
  • Q: Suppose \(U\) represents monotonic + convex preference (\(U\) is strictly increasing in all goods), is it possible that a good is inferior?

Luxury and Necessity (都属于正常商品)
