新国际经济学的伟大时代（raw）

本文最后更新于：2 个月前

本笔记整理自清华大学五道口金融学院鞠建东教授的讲座。（未整理）

一、突破性理论创新是时代的产物

理论创新：时代、运气、天赋、努力

二、当代国际经济学的理论的三大困境

国际贸易与国际金融分析框架存在割裂

  • 2008年时就存在问题

    • 中国selling高

  • 现在，美国制造业差是因为美元强势（美国米兰观点）

  • 经典贸易理论关注静态问题，当期贸易，涉及国家间产业结构变化

  • 国际金融关注经常项目调整，涉及到跨期决策问题

  • 要求新的分析方法（动态）

不存在最优的国际利益分配方案

  • 可以严格证明

  • 阿罗不可能定理

  • 不能排序，没法将社会偏好表达成社会效用函数，没法最优化求解

  • 潜在帕累托改进，收益大于损失，加政府补贴实现帕累托改进

    • 跨境利益分配无法做到，因为不存在世界政府，没法分配

  • 最重要的社会经济问题就是不同人、群体、国家之间的收入分配，利益冲突问题，因为社会偏好函数不存在，所以现代经济学无法回答这样的问题

国与国之间的利益博弈存在多重均衡

  • 经济学理论经常讨论冲击，但是多重均衡让问题难以讨论

  • 特朗普破坏了多边贸易

  • 国家利益冲突只能通过国家利益博弈来求解

  • 现实往往存在多个纳什均衡

  • 现有的分析框架下，无法确定那个贸易体系是现实选择

    • 无法观测其他的纳什均衡

三、问题

比较优势->国际分工->静态条件下跨国资源最优配置 - 分工+交换（比较优势）

自由贸易是好的

  • 为什么有些国家反对比较优势的国际分工
    • 因为静态的跨国资源最优配置，本身不构成国家利益的核心问题。
    • 国家利益的核心是经济增长
    • 按照比较优势的分工能保证一个国家的最优增长吗？不能
  • 为什么有些国家反对自由贸易？
    • 规模经济
      • 传统比较优势的分工，不一定是最优配置了

规模经济 - Helpman-Krugman model - 两部门Ricardian model，每个部门内部是具有规模经济的垄断竞争，解释产业内贸易 - 规模经济作为比较优势补充 - 行业与行业结构由比较优势决定 - 行业内部看规模经济和垄断竞争

  • 如果把规模经济拿出来，单独作为力量呢？
    • 没有解释规模经济下，比较优势所决定的国际分工还能实现资源最优配置吗
    • 为什么美国要制造业回流？
      • 比较优势与规模经济，双支柱

新国际经济学理论

  • 破解全球化新难题
  • 一个中心、两个基本点：增长为中心，比较优势+规模经济为基本点
    • 结论：自由贸易并不会帮助穷国增长，所以理应反对
  • 创建“动态结构分析”模型，将国际贸易的静态分析与国际金融的动态分析统一在同一个分析框架下，以经济增长而非贸易模式作为“新国际经济学”的分析核心
  • 建立国家之间“长期、整体、短期、局部”利益冲突的贸易争端分析框架，全球价值链与国际货币体系的结构“匹配—失衡”模型，大国竞争的“增长—危机”模型，从而分析国家之间利益冲突、大国竞争、全球治理与世界秩序的演变

四、新理论的三个标准：说得通、有大用、有传承

说得通：内部逻辑是否自洽？ 有大用：能否扛得起事实、数据检验？是否具有实际应用价值？ 有传承：能否接得上学术血脉？是否建立在已有研究基础上？或是启发了后续的理论发展？

五、米兰报告解读

问题诊断：现有的国际贸易和金融体系对美国不利，导致美元被过度高估、进而使得制造业衰落，贸易逆差扩大 政策目标：通过重构全球贸易和金融体系 - 扭转经常账户赤字，减少贸易逆差 - 复兴美国制造业 - 提高国际竞争力

政策工具： - 关税：用来增加财政收入、保护国内产业，作为与他国谈判的筹码 - 汇率政策：“海湖庄园协议”？美元贬值，以提高美国产品的出口竞争力 - 安全工具（军事）：

按顺序实施

  • 逻辑有漏洞，学术传承也不好，但有用

当前国际研究状态

说得通，有传承，但没用！ - X%以上的国际经济研究

六、《大国竞争与世界秩序重构》理论体系

好书，买吧

镜像策略是博弈的均衡 - 人不犯我我不犯人，人若犯我我必犯人

试图构建世界秩序的理性主义（Rationalism）分析框架

国际贸易争端三大原理

  • 自由贸易不能实现最优经济增长

  • 比较优势原理

  • 跨国垄断利润原理

  • 分工固化的增长陷阱原理

    • 自由贸易对穷国不好
    • 证明：比较封闭经济增长和自由贸易的增长，看哪个快
      • 索罗模型，增长根据生产函数
        • 技术相同前提，各国增长和生产水平会趋同稳态，与初始条件不同
          • 理由：资本报酬递减原理
      • 自由贸易情况下，这个原理被打破
        • 要素禀赋模型：要素价格均等化定理
        • 穷国与富国资本报酬一样，所以穷国没有更高储蓄更高增长
          • 初始条件决定穷国与富国增长
          • 由于国际分工，穷国恒穷，富国恒富，或者追赶速度慢
    • 塞姆尔森—蒙代尔问题（文一问）
      • 观察不到资本流动，不需要
      • 产品贸易代替了资本贸易

  • 世界银行：长期单一生产

    • 大部分发展中国家没开展工业革命。
      • 自由贸易使他们陷在单一生产中

  • 动态赫克希尔—俄林模型的例子

但美国不喜欢自由贸易，原因：规模经济

  • 1980s、90s，世界经济规模小，制造业规模经济小，发达国家制造业外包，本国经济“升级”为服务业
  • 2010年后，世界经济规模大， 制造业规模经济大，发达国家发现制造业空虚化制约了经济增长，要求制造业回流
    • 靠市场没法改变，只能靠国家政策

新经济增长理论

增长的理论问题： 外生增长理论、内生增长理论，都和问题没法联系到一起

新经济增长理论：增长理论中加入微观基础和中观基础 - 森林比喻：增长理论只描述了森林大或者小，但是没说怎么长得（没有微观基础，做不出来）

增长的微观基础：双轮驱动模型

“创新—市场”循环 - 研发投入 - 技术创新 - 市场规模 - 垄断利润 - 一定是垄断利润，创新产品一定要有垄断利润，否则为什么要创新 - 目前的增长模型，索罗是完全竞争，还有垄断竞争 - 文一argue：把垄断放进增长模型不重要，是把市场规模放进增长模型重要 - 关键点就是把垄断利润作为微观基础 - （循环）

  • 强调创新，同时强调市场规模的力量
    • 考虑微观基础时，只有规模才能带来垄断利润
    • 文一 argue：产能都过剩，市场规模带来价格降低
  • 目前：只有故事，没有理论模型

增长的中观基础：增长定理 一个经济体保持长期增长的充分必要条件是其产业结构的持续升级。

充分必要条件，可以严格证明，有文章

大国竞争的“增长—危机”模型

中国恰恰没有遵守自由贸易，开放经济同时保证产业结构升级，所以中国经济发展了。 - 开放的成功，但不是自由贸易的成功

大国竞争输赢，用长期增长率代表，用产业结构升级代表

赢：达到（最大）潜在增长率（A）

多重均衡的长期均衡解（双赢、双输、赢输、输赢）

中国的危机与防范

危机：多重均衡中的坏均衡

美国的危机与防范

解释特朗普为什么反对自由贸易

我们可以支持自由贸易，但是让美国同意自由贸易很难
