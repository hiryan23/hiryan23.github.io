基本概念

  • Objects: nodes, vertices
    • \(V, n=|V|\)
  • Interactions: links, edges
    • \(E, m=|E|\)
  • Systems: networks, graphs
    • \(G(V,E)\)

点（vertex）

度

  • 无向图
    • 平均度：\(\cfrac{2m}{n}\)
  • 有向图
    • 出度
    • 入度
    • 平均入度=平均出度

边（edge）

  • Weight
  • Ranking
  • Type
  • Properties depending on the structure of the rest of the graph: Number of common friends

二分图（Bipartite Graph）

连通性（connectivity）

  • 强连通有向图
    • 每个点到其它所有点都有路径
    • 强连通分量
  • 弱连通有向图
    • 如果忽略边的方向，是连通的
  • 非连通图，邻接矩阵可以拆成分块

图的数据结构

图的存储 - OI Wiki (oi-wiki.org)

邻接矩阵（Adjacency Matrix）

\[\begin{cases} A_{ij}=1\ \ \ \ \text{if there is a link from vertex $i$ to vertex $j$}\\ A_{ij}=0\ \ \ \ \text{Otherwise} \end{cases}\]

  • 有向图不一定是对称的

邻接表（Adjacency list）

存每个点和其出边的终点 - 更好服务于大、稀疏的网络

稀疏矩阵行压缩（Compressed Sparse Row, CSR）

  • 例子 \[\begin{bmatrix} 1 &1 &0 &0 \\ 0 &1 &1 &0 \\ 1 &0 &1 &1 \\ 0 &1 &0 &1 \end{bmatrix}\] \[\begin{align} \text{edge-array}&=[0,1,1,2,0,2,3,1,3]\\ \text{vertex-array}&=[0,2,4,7,9] \end{align}\]
